Please generate a personal access token for your GitHub account and add this to your ~/.m2/settings.xml:
```
   </servers>
      <server>
         <id>sapl</id>
         <username>USERNAME</username>
         <!-- see https://github.com/settings/tokens -->
         <password>ACCESS TOKEN</password>
      </server>
   </servers>
```
